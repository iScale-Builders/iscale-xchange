# Security Policy

## Reporting a Vulnerability

Please report security issues privately by emailing contact@iscalelabs.com.

Include a clear description, reproduction steps, affected URLs or files, and any
supporting screenshots or logs. Do not open public issues for vulnerabilities.

We will review reports as quickly as possible and coordinate a fix before public
disclosure when the issue is valid.

## Scope

This policy covers the iScaleXchange public web application and repository.

Out of scope:

- Social engineering
- Denial-of-service testing
- Automated scans that create excessive traffic
- Issues in third-party services unless the iScaleXchange integration is the
  source of the vulnerability
